What is GitHub?

GitHub is a web-based platform that provides hosting for softwares development projects using Git, enabling version control, collaboration, and sharing of code among developers. Git keeps track of changes in code, so you can see the history, undo mistakes and manage different versions of a project.

In a nutshell, GitHub is a platform of version control and collaboration that lets developers store, share, and work on code together.

The Interface of GitHub

Main GitHub Dashboard (after logging in)

  • Naviation Bar (Top) – Includes links to search, notifications, profile, and settings.

  • search Bar – Lets you search for repository, users, or code.

  • Repository List – Shows you own repos and ones you have contributed to.

  • Activity Feed – Updates about repos, commits, pull request, and issues from projects you follow.

Inside a Repository

  • Code Tab – Displays flies and folders of the project. You can browse code, read documentation, or clone/download it.

  • Issues Tab – Track bugs, tasks, and feature requests.

  • Pull Requests Tab – Where changes are proposed, reviewed, and merged.

  • Actions Tab – Shows automated workflows (like tests or deployments).

  • Projects Tab – Boards for organizing tasks (like trello).

  • Wiki Tab – Documentation or guides for the project.

  • Security Tab – Vulnerability alerts, dependency reports.

  • Insights Tab – Analytics about repo activity, contributors, and traffic.

  • Notification Tab – Helps you stay on top of important project changes. It keeps you updated.

  • Explore Tab – Lets you discover projects, topics, and trending repository. It helps you discover new projects and communities.

  • Settings Tab – Manage repo details (name, access, collaborators, etc)

File View

  • Displays file contents.

  • Options for editing (if you have permission), viewing history, or seeing who changed what.

Profile Page

  • Shows your bio, pinned repos, contribution graph (green squares), and activity.

Basically, the interface is organized into dashboards, repos, and tabs that cover code, collaboration, and project management.

How to Create a Repository on GitHub

After logging in

  • Click the "+" icon at the top right (new repository)
  • Enter your repository name (e.g new-repository)
  • Description is optional
  • Choose public or private
  • Click Create repository and then you are done

How to collaborate on GitHub

  1. Open your repository on GitHub
  2. Go to settings, click on collaborators
  3. Click Add collaborator and enter their GitHub username
  4. They accept the invite you both push/pull changes.
